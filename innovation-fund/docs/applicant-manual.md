# 학생 지원금 신청 플랫폼 — 신청자 이용 매뉴얼

> **최종 업데이트일**: 2026-07-07 · **플랫폼 버전**: v1.1.179 기준
> **대상 사이트**: https://phi-sage-18.vercel.app/
> 본 매뉴얼의 메뉴명·버튼명·입력항목명은 실제 화면(배포 코드)과 동일한 표기입니다.
> 프로그램별 작성 항목·제출 서류는 사이트의 **[이용안내] → 「신청 방법 — 유형별 상세 안내」** 에서 실시간(현재 등록된 신청폼 기준)으로 확인할 수 있습니다.

## Ⅰ. 플랫폼 개요

### 1. 플랫폼 목적
강원대학교 데이터보안·활용 혁신융합대학사업단이 운영하는 **학생 지원금 신청 및 공간대여 플랫폼**입니다. 지원금 신청 접수, 검토·심의, 지급 관리와 공간(강의실 등) 대여 신청을 온라인으로 처리합니다.

### 2. 이용 대상
- 강원대학교 재학생(학부) — 기본 대상
- 대학원생 — **근로장학금에 한하여** 지원 가능
- 자격증 취득 우수성과 지원금 — **미래융합가상학과 학생에 한함**

지급 제한(신청 불가): 대학원생(근로장학금 제외)·대학원 수료생·연구과정 참여학생, 휴학생·졸업생·졸업유예생, 학기 미등록자, 징계 처분 중인 자, 허위 신청자·지원조건 미이행자, 사업 목적과 무관한 활동 신청자

### 3. 신청 가능한 항목 (메뉴별 이용 목적)

| 상단 메뉴 | 이용 목적 | 하위 신청 유형 |
|---|---|---|
| **지원신청** | **활동 시작 전** 참여를 사전 신청 | 근로장학금 · 프로그램 참여지원비 · 진행요원비 |
| **지원금신청** | **활동/이수 후** 실제 지원금 지급 신청 | 근로장학금 · 프로그램 참여지원비 · 진행요원비 · 성적 우수 지원금 · 경진대회 입상 우수성과 지원금 · 자격증 취득 우수성과 지원금 |
| **소학회** | 첨단 ICT 소학회(동아리) 활동 지원 | 지원신청 / 지원금신청 |
| **공간대여** | 공간(강의실 등) 대여 신청·이용결과 제출 | 공간대여 신청 · 이용결과 제출 (로그인 불필요) |

### 4. 지원신청과 지원금신청의 차이

| 구분 | 지원신청 | 지원금신청 |
|---|---|---|
| 신청 시점 | **활동 시작 전** | **활동/이수 완료 후** |
| 목적 | 참여 자격 사전 승인 | 실제 지원금 지급 요청 |
| 대상 유형 | 근로장학금·프로그램 참여지원비·진행요원비(·소학회) | 왼쪽 3종 + 성적/경진대회/자격증(성과형) |
| 결과 | **승인**되면 활동 후 지원금신청 가능 (마이페이지에 [지원금 신청하기] 버튼 표시) | 검토 후 지급 처리 |
| 연계 | — | 승인된 지원신청 내역 선택 시 **중복 항목 자동 입력** |

### 5. 신청 전 반드시 확인해야 할 사항
가. 해당 유형의 **신청 기간**이 열려 있는지(홈 화면 유형 카드에 「신청중 / 기간 아님」 표시)
나. 본인이 **지원 대상**인지(위 2항)
다. 활동 전이면 **지원신청**, 활동 후면 **지원금신청**인지
라. **본인 명의 계좌**와 **통장 사본**(예금주·계좌번호 일치) 준비 여부

[화면 캡처 삽입: 홈 화면 — 유형 카드와 「신청중」 표시]
※ 설명 문구: 홈 화면의 지원금 유형 카드입니다. 카드를 클릭하면 지급 기준 상세가 열리며, 신청 기간 여부가 표시됩니다.

## Ⅱ. 신청 전 준비사항

### 1. 준비 항목
| 구분 | 준비 내용 |
|---|---|
| 가. 본인 정보 | 이름, 학번, 연락처(010-0000-0000), 이메일 |
| 나. 학적 정보 | 학적 유형(대학생/대학원생), 캠퍼스, 단과대학, 학과(대학원생: 대학원 종류·전공) |
| 다. 계좌 정보 | 은행, 예금주, 계좌번호(‘-’ 없이) — **반드시 본인 명의** |
| 라. 증빙서류 | 유형별 상이(아래 표) — 통장 사본은 공통 권장 |
| 마. 서명 | 화면에 직접 그리거나 서명 이미지 파일(PNG·JPG) |
| 바. 개인정보 동의 | 신청 단계에서 필수 체크(미동의 시 진행 불가) |

### 2. 파일 형식 및 업로드 전 확인사항
- 업로드 가능 형식: **PDF · JPG · PNG · WEBP** (그 외 형식은 거부됨)
- 사진 촬영본은 글자가 선명하게 보이는지 확인
- 항목에 안내 문구가 있으면 업로드 직전 확인창이 표시됨(예: 직인 날인본 요구)

### 3. 신청 유형별 신청 시점·준비서류 표

| 신청 유형 | 신청 시점 | 준비서류(대표) |
|---|---|---|
| 근로장학금 | 지원신청(활동 전) → 지원금신청(근무 후) | 통장 사본, 근무상황부(플랫폼 입력), 수강 시간표(마이페이지 등록) |
| 프로그램 참여지원비 | 지원신청 → 지원금신청 | 통장 사본, 참가확인서·결과보고 등 증빙, (해당 시) 등록비·교통비·숙박비 영수증 |
| 진행요원비 | 지원신청 → 지원금신청 | 통장 사본, 활동 증빙 |
| 성적 우수 지원금 | 지원금신청(이수 후) | 성적/이수 확인 서류(안내에 따름), 통장 사본 |
| 경진대회 입상 우수성과 지원금 | 지원금신청(입상 후) | 입상 증빙(상장·결과), 통장 사본 |
| 자격증 취득 우수성과 지원금 | 지원금신청(취득 후) | 자격증 사본, 통장 사본 ※ 미래융합가상학과 학생만 |
| 소학회 | 지원신청 → 지원금신청 | 구성원 명단(회장+팀원, 최소 6명), 활동 증빙 |

> ※ 프로그램마다 관리자가 설정한 신청폼이 달라 실제 항목·서류가 다를 수 있습니다. **[이용안내]의 「신청 방법 — 유형별 상세 안내」** 에서 현재 기준으로 확인하세요. (신청폼이 바뀌면 해당 안내가 자동으로 갱신됩니다)

## Ⅲ. 회원가입 또는 로그인 방법

### 1. 접속 방법
① 브라우저 주소창에 플랫폼 주소를 입력합니다.
② 홈 화면 우측 상단의 **[로그인]** 을 클릭합니다. (지원금 신청은 로그인 필수 / **공간대여는 로그인 없이 가능**)

### 2. 회원가입 방법
[화면 캡처 삽입: 로그인 화면 — 「회원가입」 탭]
※ 설명 문구: 로그인 화면에서 [회원가입] 탭을 누르면 가입 양식이 나타납니다. 아이디는 학번입니다.

① [로그인] 화면 상단의 **[회원가입]** 탭을 클릭합니다.
② **학번**, **이름** 을 입력합니다.
③ **비밀번호(8자 이상)**, **비밀번호 확인** 을 입력합니다.
④ **학적 유형**(대학생/대학원생)을 선택합니다.
⑤ **캠퍼스** 를 선택하고, 대학생은 **단과대학·학과**(검색·직접입력 가능), 대학원생은 **대학원 종류·단과대학·학과·전공** 을 입력합니다.
⑥ **연락처**, **이메일** 을 입력합니다.
⑦ **계좌 정보(지원금 입금 계좌)** — 은행·예금주·계좌번호를 입력합니다.
⑧ 개인정보 수집·이용 안내를 읽고 **동의** 에 체크합니다. [필수]
⑨ **[회원가입 후 시작하기]** 를 클릭합니다.

### 3. 로그인 방법
① [로그인] 탭에서 **학번** 을 입력합니다.
② **비밀번호** 를 입력합니다.
③ **[로그인]** 버튼을 클릭합니다.

### 4. 비밀번호 관련 유의사항
- 아이디는 **학번**입니다.
- **비밀번호 찾기 기능이 없습니다.** 분실 시 사업단에 문의하면 관리자가 재설정해 안내합니다.
- 로그인 후에는 **마이페이지 → [비밀번호 변경]** 에서 직접 변경할 수 있습니다(변경 후 재로그인 필요).

### 5. 정보 오입력 시 대처 방법
| 상황 | 대처 |
|---|---|
| 이름·연락처·이메일·계좌 오입력 | 마이페이지 → **[개인정보 수정]**(비밀번호 본인 확인 후 수정) |
| 학번이 바뀜(대학원 진학 등) | 마이페이지 → **[학적상태변경]** — 기존 신청 기록은 유지되며 새 학번으로 재로그인 |
| 학번 자체를 잘못 가입 | 사업단 문의(관리자 처리) |

## Ⅳ. 홈 화면 이용 방법

[화면 캡처 삽입: 홈 화면 전체]
※ 설명 문구: 상단 메뉴(지원신청·지원금신청·소학회·공간대여), 지원금 유형 카드, 공간대여 예약 현황, 우측 바로가기(이용안내 등)로 구성됩니다.

### 1. 운영 절차 확인 방법
홈 화면의 운영 절차 안내: **모집 공고 및 안내 → 신청 및 접수 → 검토 및 심의 → 대상 확정 및 통보 → 지원금 지급**

### 2. 유형별 지급 기준 확인 방법
① 홈 화면에서 해당 **지원금 유형 카드** 를 클릭합니다.
② 열리는 창에서 지급 금액·조건·세부 기준을 확인합니다. (예: 성적 우수 — 마이크로디그리 30만원 · 부전공 100만원 · 복수전공 150만원, 평점 평균 3.0 이상)

### 3. 지급 대상·지급 제한 확인 방법
홈 화면의 지급 대상/제한 안내 문단에서 확인합니다(본 매뉴얼 Ⅰ-2 동일).

### 4. 공간대여 예약 현황 확인 방법
홈 화면의 **공간대여 캘린더** 에서 월별 예약 현황을 확인합니다. (상단 메뉴 [공간대여]에서도 동일 확인 가능)

### 5. 문의처 확인 방법
- 화면 우측(모바일: 하단) 바로가기의 **[카톡 문의]**
- 하단 푸터: 이메일 **sducoss@kangwon.ac.kr** · 전화 **033-250-7879** · 위치 **강원대학교 한빛관 1층 105호**
- 우측 바로가기 **[이용안내]** — 새 탭으로 신청 방법 상세 안내가 열립니다.

## Ⅴ. 지원신청 이용 방법

> **지원신청은 “활동을 시작하기 전”에 하는 사전 신청입니다.** 승인되어야 활동 후 지원금신청으로 이어집니다.

### 1. 메뉴 접근 경로
① 홈 상단 메뉴에서 **[지원신청]** 을 클릭합니다. (마우스를 올리면 하위 유형 목록이 펼쳐집니다)
② 섹션 페이지에서 유형별 **[지원하기]** 버튼을 클릭합니다.

[화면 캡처 삽입: 지원신청 섹션 페이지(/menu/pre)]
※ 설명 문구: “활동 시작 전, 참여할 지원 종류를 선택해 신청합니다.” 유형별 신청기준 안내와 [지원하기] 버튼이 표시됩니다.

### 2. 신청 유형 선택 방법
지원신청 가능 유형: **근로장학금 / 프로그램 참여지원비 / 진행요원비** (3종)

### 3. 유형별 지원신청 방법 (공통 흐름)
① 유형을 선택하면 **프로그램명** 선택 화면이 나타납니다. “프로그램을 선택하세요” 드롭다운에서 참여할 프로그램을 고릅니다.
② 선택한 프로그램의 신청서 양식(단계별 폼)이 표시됩니다.
③ **[기본 정보]** 단계 — 이름·학번·소속 등은 회원정보에서 **자동 입력**됩니다. 확인 후 [다음 단계].
④ **개인정보 수집·이용 및 신청 동의** — 필수 항목에 모두 체크합니다.
⑤ 프로그램별 작성 항목(활동 기간·역할·계획 등)을 입력합니다.
⑥ 증빙 항목이 있으면 파일을 업로드합니다.
⑦ 서명 항목이 있으면 **직접 서명** 또는 **서명 이미지 업로드** 로 서명합니다.
⑧ 마지막 **신청내용 확인** 단계에서 검토 후 **[지원신청 제출]** 을 클릭합니다.

- **근로장학금**: 마이페이지 **[수강 시간표 (근로장학금)]** 를 먼저 등록해 두세요. 지원금신청 시 근무상황부에서 **수업시간과 겹치는 근로는 등록되지 않습니다.**
- **프로그램 참여지원비 / 진행요원비**: 프로그램·역할 선택 후 동일 흐름입니다.

### 4. 입력항목별 작성 요령
| 항목 | 요령 |
|---|---|
| 한 줄 입력/서술형 | 글자수 제한(최소/최대)이 표시되면 준수 |
| 날짜/기간 | 시작~종료 모두 선택(기간형) |
| 드롭다운 | 선택값에 따라 **추가 질문**이 열릴 수 있음 — 함께 작성 |
| 표 | 빈 칸을 모두 채움 |
| 파일 업로드 | PDF·JPG·PNG·WEBP만 |
| 파일 다운로드 | 사업단 제공 양식을 내려받아 작성 후, 업로드 항목에 첨부 |
| 서명 | [직접 서명](마우스·터치) 또는 [이미지 업로드] |

### 5. 제출 전 확인사항 (→ Ⅺ 체크리스트)
### 6. 신청 완료 후 확인 방법
제출 즉시 **접수번호** 가 발급되고, **마이페이지** 신청 내역에 「검토: 신청완료」 상태로 표시됩니다.

### 7. 신청 후 수정 또는 취소가 필요한 경우
| 상황 | 방법 |
|---|---|
| 작성 중 중단 | **[임시저장]** → 마이페이지 「임시저장(작성 중)」에서 **[이어서 신청]** ※ 임시저장은 접수가 아님 |
| 제출 후 수정 | 관리자가 **보완요청** 처리 시 마이페이지에서 **[수정 후 재제출]** 가능. 그 외에는 취소 후 재신청 또는 사업단 문의 |
| 취소 | 마이페이지 신청 카드의 **[신청 취소]** (되돌릴 수 없음) |

**주의사항**
- 임시저장만 하고 제출하지 않으면 **접수되지 않습니다.**
- 지원신청이 **승인되기 전에는** 지원금신청으로 이어갈 수 없습니다(마이페이지에 “관리자 승인 후 지원금 신청이 가능합니다” 표시).

## Ⅵ. 지원금신청 이용 방법

> **지원금신청은 “활동/이수가 끝난 후” 실제 지원금 지급을 요청하는 신청입니다.**

### 1. 메뉴 접근 경로
① 홈 상단 메뉴 **[지원금신청]** 클릭 → 섹션 페이지에서 유형별 **[지원금 신청하기]** 클릭
② 또는 마이페이지에서 승인된 지원신청 카드의 **[지원금 신청하기]** 클릭(중복 항목 자동 입력)

[화면 캡처 삽입: 지원금신청 섹션 페이지(/menu/fund)]
※ 설명 문구: “활동 후, 신청할 지원금 종류를 선택해 신청합니다.” 6개 유형이 표시됩니다.

### 2. 유형별 신청 방법

**가. 근로장학금**
① 유형 선택 → (승인된 지원신청 내역이 있으면 선택: 자동 입력 / 없으면 **[지원신청 내역 없이 새로 작성]**)
② 프로그램 선택 → 기본정보 확인 → 동의 체크
③ **근무상황부**: 근무일자·시작·종료 입력 후 **[＋ 등록]** — 근무시간이 자동 합산되고 단가에 따라 지급액이 자동 계산됩니다. (1일 최대 8시간, 수업시간과 겹치면 등록 불가)
④ 서명 → 신청내용 확인 → **[신청 제출]**

**나. 프로그램 참여지원비**
① 위와 동일 흐름으로 프로그램 선택
② 활동 내용·성과 작성, (해당 시) **등록비**(금액 입력) · **교통비**(일자·수단·금액·출발/도착·행별 증빙) · **숙박비**(결제금액·증빙, 1인 70,000원 한도) 입력
③ 증빙 서류 업로드 → 계좌 확인 → 서명 → 제출

**다. 진행요원비** — 프로그램·역할 선택 후 활동 내용 작성, 증빙 업로드, 제출(동일 흐름).

**라. 성적 우수 지원금**
[화면 캡처 삽입: 성적 우수 지원금 — 세부 유형·교과목 입력]
※ 설명 문구: 세부 유형 선택 후 이수 교과목·학점·성적을 입력하면 평점·인정학점·지급 예정액이 자동 계산됩니다.

① 세부 유형 선택: **마이크로디그리(30만원) / 부전공(100만원) / 복수전공(150만원)**
② (마이크로디그리) 학과 → MD 과정 선택 → 과정 교과목별 성적 입력
③ (부전공/복수전공) “미래융합가상학과 부전공/복수전공 이수(예정)자임을 확인합니다” 체크 → 전공 선택 → **[+ 과목 추가]** 로 이수 교과목 입력
④ 교과목이 목록에 없으면 드롭다운에서 **✏️ 직접입력** 선택 후 교과목명을 직접 입력
⑤ MD 이수과목은 「MD 과정」 지정, 인정 제외 과목은 **[학점 불인정]** 체크(MD는 4과목 중 3과목만 학점 인정)
⑥ **지원 자격 확인** 체크리스트(평점 3.0 이상·인정 학점 기준·MD 1개 이상 등)가 모두 충족되면 제출
※ ‘가/부’ 과목은 평점 평균 계산에서 제외됩니다.

**마. 경진대회 입상 우수성과 지원금** — 대회명·입상 내역 입력, **입상 증빙(상장·결과)** 업로드 후 제출. (A규모·B규모/개인·팀별 차등 지급)

**바. 자격증 취득 우수성과 지원금** — ※ **미래융합가상학과 학생만**. 자격증명·취득일 입력, **자격증 사본** 업로드 후 제출. (난이도 상 70 · 중 40 · 하 10만원) 지원 대상 자격증은 홈 화면 **자격증 목록** 에서 확인.

### 3. 지원금 유형별 신청 조건·증빙서류 — Ⅱ-3 표 참조

### 4. 계좌정보 입력 시 주의사항
- **반드시 본인 명의 계좌**만 지급됩니다.
- 제출하는 **통장 사본의 예금주·은행·계좌번호와 일치**해야 하며, 다르면 지급이 보류될 수 있습니다.
- 예금주와 신청자 이름이 다르면 화면에 경고가 표시됩니다.
- 실제 지급은 연구통합관리시스템(학생용) 등록 본인 계좌로 처리되므로 **본인계좌 등록** 여부도 확인하세요.

### 5. 신청 완료 후 처리 절차
접수(접수번호 발급) → 검토(신청완료 → 검토중) → 필요시 보완요청/심의 → **승인/반려** → 지급 처리(지급대기 → 지출결의중 → 지출완료). 상태는 **마이페이지** 에서 확인합니다.

### 6. 반려 또는 보완 요청 시 대응 방법
| 상태 | 대응 |
|---|---|
| **보완요청** | 마이페이지 상단 배너/카드의 안내(사유)를 확인 → **[수정 후 재제출]** 클릭 → 수정 후 다시 제출 |
| **반려** | 카드의 「안내」(반려 사유) 확인 → 필요 시 사업단 문의 → 조건 충족 후 재신청 |

## Ⅶ. 소학회 신청 방법

### 1. 소학회 메뉴 설명
홈 상단 **[소학회]** 메뉴 — 첨단 ICT 분야(보안·클라우드·블록체인 등) 소학회(동아리) 활동 지원. 하위에 **[지원신청]**(활동 전) / **[지원금신청]**(활동 후) 두 버튼이 있습니다.

### 2. 지원신청과 지원금신청의 차이 — Ⅰ-4와 동일(소학회도 같은 구조)

### 3. 신청 대상·구성 요건
- 첨단 ICT 분야 소학회
- **소학회별 최소 6명** 구성 · **회장은 혁신인재지원금** 대상

### 4. 회장 및 구성원 정보 입력 방법
① [소학회] → [지원신청] 또는 [지원금신청] 클릭 → 프로그램(소학회 모집) 선택
② **소학회 구성원** 항목에서 회장+팀원 명단을 입력합니다: **구분(회장/팀원) · 이름 · 학번 · 소속학과(가상학과 여부) · 연락처**
③ 운영계획/활동계획 등 서술 항목을 작성합니다.

### 5. 운영비·혁신인재지원금 신청 시 주의사항
- 구성원 명단은 실제 활동 인원과 일치해야 합니다(최소 6명).
- 활동 증빙(사진·결과보고 등)은 지원금신청 단계에서 요구될 수 있습니다.

### 6. 신청 후 확인사항
마이페이지에서 접수·검토 상태를 확인하고, 지원신청 승인 후 활동을 진행한 뒤 [지원금신청]으로 이어갑니다.

| 종료가 시작보다 빠름 | 날짜·시간 재확인(여러 날이면 종료일 지정) |
| 연락처 오입력 | 이용결과 조회가 안 됨 — 신청 때 번호를 정확히 입력 |
| 이용결과 미제출 | 사용 후 반드시 [이용결과 제출] (이후 대여에 불이익 가능) |

## Ⅸ. 신청 상태 확인 및 사후관리

[화면 캡처 삽입: 마이페이지 — 신청 내역·상태 배지]
※ 설명 문구: 신청별 「검토 상태」·「지급 상태」 배지와 진행 단계 바가 표시됩니다.

### 1. 신청 완료 여부·접수번호 확인
① 제출 직후 완료 화면에서 **접수번호** 확인
② 상단 **[마이페이지]** → 신청 내역 카드에서 접수번호·상태 확인 (상태가 안 바뀌면 **[새로고침]**)

### 2. 상태 의미
| 검토 상태 | 의미 / 할 일 |
|---|---|
| 신청완료 | 접수됨 — 검토 대기 |
| 검토중 | 담당자 검토 중 — 대기 |
| **보완요청** | 안내 확인 후 **[수정 후 재제출]** |
| 심의필요 | 위원회 심의 대상 — 별도 요청 없으면 대기 |
| 승인 | (지원신청) **[지원금 신청하기]** 진행 가능 |
| 반려 | 사유 확인 — 필요 시 문의 |

지급 상태: 지급대기 → 지출결의중 → 지출완료 (지급보류·환수대상 시 안내 확인)

### 3. 보완 요청 확인·수정 제출
① 마이페이지 상단 주황 배너 “보완 요청이 있는 신청이 N건 있습니다” 확인
② 해당 카드의 안내(사유) 확인 → **[수정 후 재제출]** → 수정 → 제출

### 4. 반려 사유 확인
카드의 「안내」 메모(관리자 작성 사유)를 확인합니다. (지원신청 반려 시 [사유 보기] 버튼)

### 5. 최종 승인 후 절차
- 지원신청 승인 → 활동 후 **[지원금 신청하기]**
- 지원금신청 승인 → 「최종 승인액」 확인 → 지급 상태가 **지출완료** 로 바뀌면 입금 확인

### 6. 문의가 필요한 경우 준비해야 할 정보 (문의 전 확인사항)

## Ⅹ. 신청자 자주 묻는 질문 (FAQ)

**Q1. 지원신청과 지원금신청 중 무엇을 선택해야 하나요?**
A. 활동을 **아직 시작하지 않았다면 [지원신청]**, 활동/이수가 **끝났다면 [지원금신청]** 입니다. 성적·경진대회·자격증(성과형)은 지원금신청만 합니다.

**Q2. 신청 전에 활동을 시작해도 되나요?**
A. 지원신청 대상 유형(근로·프로그램·진행요원)은 **활동 전에 지원신청을 먼저** 하는 것이 원칙입니다. 승인 전에 활동을 시작하면 지원이 제한될 수 있으니 사업단 공고를 확인하세요.

**Q3. 활동이 끝난 후에는 어디에서 신청하나요?**
A. 상단 **[지원금신청]** 메뉴, 또는 마이페이지의 승인된 지원신청 카드에서 **[지원금 신청하기]**(자동 입력)로 신청합니다.

**Q4. 첨부파일을 잘못 올렸습니다.**
A. 제출 전이면 업로드 목록의 파일 삭제(✕) 후 다시 업로드하세요. 제출 후에는 관리자가 **보완요청** 처리 시 재제출할 수 있습니다 — 사업단에 문의하세요.

**Q5. 계좌정보를 잘못 입력했습니다.**
A. 마이페이지 **[개인정보 수정]** 에서 계좌를 수정하세요(다음 신청부터 반영). 이미 제출한 신청 건은 사업단에 문의하면 확인 후 처리됩니다.

**Q6. 신청서를 제출했는데 수정하고 싶습니다.**
A. 상태가 검토 전/보완요청이면 수정 여지가 있습니다. **보완요청** 상태면 [수정 후 재제출], 아니면 [신청 취소] 후 재신청하거나 사업단에 문의하세요.

**Q7. 보완 요청을 받았습니다.**
A. 마이페이지에서 안내(사유)를 확인하고 **[수정 후 재제출]** 로 해당 부분을 고쳐 다시 제출하면 됩니다.

**Q8. 반려되었습니다.**
A. 카드의 「안내」에서 반려 사유를 확인하세요. 조건 미충족이면 충족 후 재신청, 판단에 이의가 있으면 접수번호와 함께 사업단에 문의하세요.

**Q9. 공간대여 신청 후 이용결과는 꼭 제출해야 하나요?**
A. 네. 공간 사용 후 **[이용결과 제출]** 에서 이용자 명단·서명과 이용 사진을 제출해야 합니다.

**Q10. 모바일에서도 신청할 수 있나요?**
A. 네, 모바일 브라우저에서 이용 가능합니다(반응형). 서명은 손가락으로 그릴 수 있습니다. 파일 업로드·표 입력이 많은 신청은 PC 이용을 권장합니다.

**Q11. 문의할 때 어떤 정보를 알려야 하나요?**
A. Ⅸ-6 표(접수번호·이름·학번·신청 유형·프로그램명·오류 화면)를 준비해 주세요.

## Ⅺ. 신청 전 최종 체크리스트

### 1. 공통
- [ ] 로그인(학번) 완료 (공간대여 제외)
- [ ] 신청 기간 「신청중」 확인
- [ ] 지원 대상 해당 여부 확인
- [ ] 개인정보 동의 체크
- [ ] 필수(*) 항목 모두 작성
- [ ] 서명 완료(“서명이 적용되었습니다” 표시)
- [ ] **[제출]까지 완료** (임시저장은 접수 아님)
- [ ] 접수번호 확인·메모

### 2. 지원신청
- [ ] 활동 **시작 전**인지 확인
- [ ] 프로그램·역할 정확히 선택
- [ ] (근로장학금) 마이페이지 수강 시간표 등록

### 3. 지원금신청
- [ ] 활동/이수 **완료 후**인지 확인
- [ ] (연계 유형) 승인된 지원신청 내역 선택 여부
- [ ] 증빙서류(영수증·확인서·상장·자격증 등) 업로드 — PDF·JPG·PNG·WEBP
- [ ] 통장 사본과 계좌정보(예금주·계좌번호) 일치
- [ ] (근로) 근무상황부 등록 완료 / (성적) 자격 확인 체크리스트 전부 충족

### 4. 소학회
- [ ] 구성원 최소 6명(회장+팀원) 명단 입력
- [ ] 활동/운영 계획 작성
- [ ] 활동 전=지원신청 / 활동 후=지원금신청 구분

### 5. 공간대여
- [ ] 캘린더에서 겹치는 예약 없는지 확인
- [ ] 연락처 정확히 입력(이용결과 조회용)
- [ ] 사용 후 [이용결과 제출] (명단·서명·사진)

   - 필요한 이유: 프로그램·폼은 관리자가 수시로 등록/수정(DB 저장)하므로 문서 시점과 다를 수 있음

   - 매뉴얼 반영 위치: Ⅴ·Ⅵ·Ⅶ의 프로그램 예시 / ※ 플랫폼 [이용안내]의 동적 안내가 실시간 기준입니다.
2. **확인이 필요한 내용**: 유형별 지급 기준(홈 카드 클릭 시 내용)의 확정 문안
   - 필요한 이유: 관리자 편집 콘텐츠(리치텍스트)라 수시 변경 가능
   - 질문: “유형별 지급 기준 최신 확정본을 공유해 주세요.”
   - 반영 위치: Ⅳ-2, Ⅱ-3 표
3. **확인이 필요한 내용**: 성적/경진대회/자격증(성과형)의 세부 증빙서류 목록·규격
   - 필요한 이유: 화면에는 입력 양식만 있고 서류 규격(발급처·유효기간 등)은 내부 지침
   - 질문: “성과형 3종의 증빙서류 목록과 규격 기준을 알려주세요.”
   - 반영 위치: Ⅱ-3, Ⅵ-2 라~바

   - 질문: “모바일 실기기 테스트 시 불편 사례가 접수된 것이 있나요?”
