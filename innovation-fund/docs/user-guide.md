# 사용자(학생) 매뉴얼

혁신인재지원금 + 공간대여 플랫폼 — 학생(신청자)이 처음 이용해도 그대로 따라 할 수 있는 안내입니다.
각 기능은 `기능 설명 → 대상 → 접근 경로 → 사전 확인 → 단계별 방법 → 입력 항목 → 완료 확인 → 주의사항 → 오류 해결 → 관련 FAQ` 순으로 정리했습니다.

> 화면 이미지(`images/*.png`)는 캡처 자동화 스크립트로 생성됩니다([README](./README.md) 참고). 실행 전에는 이미지가 비어 있을 수 있습니다.

## 1. 회원가입

![회원가입 화면](images/signup-screen.png)

1. **기능 설명**: 플랫폼 계정을 만듭니다. 아이디는 학번입니다. 지원금 신청 전 반드시 필요합니다.
2. **사용 대상**: 강원대학교 학생(학부 재학생 기본, 대학원생 포함).
3. **접근 경로**: 홈 → 상단 [로그인] → **[회원가입]** 탭. (URL: `/login`)
4. **사용 전 확인사항**: 학번, 본인 명의 계좌(은행·예금주·계좌번호), 이메일을 준비합니다.
5. **단계별 사용 방법**
   1. 홈 우측 상단 **[로그인]** 을 클릭합니다.
   2. 상단 **[회원가입]** 탭을 클릭합니다.
   3. **학번**과 **이름**을 입력합니다.
   4. **비밀번호(8자 이상)** 와 **비밀번호 확인**을 입력합니다.
   5. **학적 유형**(대학생/대학원생)을 선택합니다.
   6. **캠퍼스**를 선택합니다. 대학생은 **단과대학·학과**(검색·직접입력 가능), 대학원생은 **대학원 종류·단과대학·학과·전공**을 입력합니다.
   7. **연락처**와 **이메일**을 입력합니다.
   8. **계좌 정보**(은행·예금주·계좌번호)를 입력합니다.
   9. 개인정보 수집·이용 안내를 읽고 **[동의]** 에 체크합니다.
   10. **[회원가입 후 시작하기]** 를 클릭합니다.
6. **입력해야 하는 항목**: 학번, 이름, 비밀번호(8자↑), 학적유형, 캠퍼스, 단과대학, 학과, (대학원생) 전공, 연락처, 이메일, 은행, 예금주, 계좌번호, 개인정보 동의.
7. **처리 완료 후 확인할 사항**: 가입 후 자동 로그인되어 홈으로 이동합니다.
8. **주의사항**: 계좌는 **본인 명의**여야 하며, 나중에 제출하는 통장 사본과 예금주·계좌번호가 일치해야 지급됩니다. 개인정보 미동의 시 가입 불가.
9. **오류 발생 시 해결 방법**: "다음 항목을 모두 입력해주세요" → 빨간 `*` 필수 항목 확인. "비밀번호는 8자 이상" / "비밀번호가 일치하지 않습니다" → 비밀번호 재입력. 이미 가입된 학번이면 로그인하세요.
10. **관련 FAQ**: [로그인/계정] 분류 참고([faq.md](./faq.md)).

## 2. 로그인

![로그인 화면](images/login-screen.png)

1. **기능 설명**: 학번·비밀번호로 로그인합니다.
2. **사용 대상**: 가입한 학생.
3. **접근 경로**: 홈 → **[로그인]** (URL: `/login`).
4. **사용 전 확인사항**: 학번과 비밀번호.
5. **단계별 사용 방법**
   1. 홈 우측 상단 **[로그인]** 클릭.
   2. **[로그인]** 탭에서 **학번** 입력.
   3. **비밀번호** 입력.
   4. **[로그인]** 버튼 클릭.
6. **입력해야 하는 항목**: 학번, 비밀번호.
7. **처리 완료 후 확인할 사항**: 원래 가려던 페이지(또는 홈)로 이동합니다.
8. **주의사항**: 아이디는 학번입니다.
9. **오류 발생 시 해결 방법**: "로그인 실패" → 학번/비밀번호 확인. 비밀번호를 잊었다면 별도 찾기 기능이 없으므로 **사업단에 문의**하여 재설정 안내를 받으세요. (로그인 후에는 마이페이지에서 변경 가능)
10. **관련 FAQ**: [로그인/계정].

## 3. 홈 화면·지원금 유형 확인

![사용자 홈 화면](images/home-user.png)

1. **기능 설명**: 지원금 8개 유형의 대상·금액·조건, 신청 기간, 공간대여 캘린더, 자격증 목록, 바로가기·이용안내를 확인합니다.
2. **사용 대상**: 누구나(공개). 신청은 로그인 필요.
3. **접근 경로**: `/` (홈).
4. **사용 전 확인사항**: 신청하려는 유형의 **신청 기간**이 열려 있는지 확인.
5. **단계별 사용 방법**
   1. 홈에서 유형 카드(예: 근로장학금·프로그램 참여지원비·성적 우수 지원금 등)를 클릭해 상세 안내를 봅니다.
   2. 유형 옆 **신청중 / 기간 아님** 표시로 신청 가능 여부를 확인합니다.
   3. 우측(또는 하단) 바로가기에서 **이용안내**·사업단 홈페이지·LMS·이루리·카톡 문의를 이용합니다.
6. **입력해야 하는 항목**: 없음.
7. **처리 완료 후 확인할 사항**: 신청 가능한 유형·기간 확인.
8. **주의사항**: **신청 기간이 아니면** 해당 유형 신청이 제한됩니다.
9. **오류 발생 시 해결 방법**: 유형이 안 보이거나 기간이 아니면 사업단 공지를 확인하세요.
10. **관련 FAQ**: [자산 조회→지원금 유형 확인] 분류.

## 4. 지원신청(사전신청)

1. **기능 설명**: 활동 시작 **전에** 참여를 신청합니다. 승인되면 활동 후 지원금 신청으로 이어집니다. (근로장학금·프로그램 참여지원비·학생활동 등에서 사용)
2. **사용 대상**: 로그인한 학생.
3. **접근 경로**: 홈 상단 메뉴 → 지원신청, 또는 `/apply?mode=pre`.
4. **사용 전 확인사항**: 해당 유형 신청 기간, 대상 자격.
5. **단계별 사용 방법**
   1. 지원신청 화면에서 **종류(유형)** 를 선택합니다(근로장학금/프로그램 참여지원비/진행요원비).
   2. (프로그램형) **프로그램**을 선택하면 해당 신청서 양식이 나타납니다.
   3. 단계별 폼을 작성하고 **[다음 단계]** 로 진행합니다.
   4. 마지막에 **[지원신청 제출]** 을 클릭합니다.
6. **입력해야 하는 항목**: 유형/프로그램, 폼 항목(프로그램별 상이), 개인정보 동의.
7. **처리 완료 후 확인할 사항**: 마이페이지에서 상태가 접수(검토 대기)로 표시.
8. **주의사항**: 지원신청이 **승인**되어야 지원금 신청을 이어서 할 수 있습니다.
9. **오류 발생 시 해결 방법**: 필수 항목 누락 시 안내창이 뜹니다. 기간 아님 → 사업단 공지 확인.
10. **관련 FAQ**: [대여 신청→지원 신청] 분류.

## 5. 지원금 신청

![신청 종류 선택](images/apply-type-select.png)

1. **기능 설명**: 활동/이수 후 지원금을 신청합니다. 유형별 신청서 양식으로 작성합니다.
2. **사용 대상**: 로그인한 학생.
3. **접근 경로**: 홈 상단 메뉴 → 지원금 신청, 또는 마이페이지 **[새 신청]**, URL `/apply`.
4. **사용 전 확인사항**: 신청 기간, (일부 유형) 지원신청 승인 여부, 증빙 서류(통장사본 등).
5. **단계별 사용 방법**
   1. **[새 신청]** 또는 홈 메뉴로 신청 화면에 들어갑니다.
   2. **종류(유형)** 를 선택합니다.
   3. (지원신청 연계 유형) 이전 **지원신청 승인 내역**이 있으면 선택하면 중복 항목이 자동 입력됩니다. 없으면 **[지원신청 내역 없이 새로 작성]**.
   4. (프로그램형) **프로그램**을 선택합니다.
   5. 단계별 폼을 작성합니다(6장 참고).
   6. **신청 내용 확인** 단계에서 검토 후 **[신청 제출]** 을 클릭합니다.
6. **입력해야 하는 항목**: 유형/프로그램, 기본정보(자동), 개인정보 동의, 폼 항목(서술·파일·서명·근무상황부·비용 등), 계좌(자동).
7. **처리 완료 후 확인할 사항**: 접수번호 발급 → 마이페이지에서 상태 확인.
8. **주의사항**: 파일은 PDF·JPG·PNG·WEBP만 업로드됩니다. 임시저장은 접수가 아닙니다.
9. **오류 발생 시 해결 방법**: "종류/기간" 문제 → 홈 확인. 승인 전 지원금 신청 불가 유형은 지원신청 승인 후 진행.
10. **관련 FAQ**: [대여 신청].

## 6. 신청서 작성 — 공통 항목

신청서는 프로그램마다 항목이 다릅니다. 자주 나오는 항목의 작성법입니다.

### 6-1. 개인정보 수집·이용 동의
![개인정보 동의](images/privacy-consent.png)
1. **기능 설명**: 신청 단계에서 개인정보 수집·이용 및 신청 동의에 체크합니다.
2. **대상**: 신청자. 3. **접근 경로**: 신청서 작성 중 해당 단계.
4. **사전 확인**: 안내문 확인.
5. **단계별**: (1) 안내문을 읽습니다. (2) **필수 동의 항목(개인정보/사실확인/본인계좌)** 에 모두 체크합니다.
6. **입력 항목**: 체크박스(모두 필수). 7. **완료 확인**: 다음 단계로 진행 가능.
8. **주의사항**: 하나라도 미체크 시 다음 단계로 넘어갈 수 없습니다.
9. **오류 해결**: "동의 항목에 모두 체크해주세요" → 미체크 항목 확인.
10. **FAQ**: [개인정보 동의].

### 6-2. 서명 입력
![서명 입력](images/signature-input.png)
1. **기능 설명**: 신청서에 서명을 넣습니다(직접 그리기 또는 이미지 업로드).
2. **대상**: 신청자. 3. **접근 경로**: 신청서 서명 항목.
4. **사전 확인**: 마우스/터치로 서명하거나 서명 이미지 파일 준비.
5. **단계별**: (1) **[직접 서명]** 탭에서 마우스·손가락으로 서명하거나, (2) **[이미지 업로드]** 탭에서 서명 이미지를 올립니다. (3) "서명이 적용되었습니다" 표시를 확인합니다.
6. **입력 항목**: 서명(그리기 또는 PNG·JPG 이미지).
7. **완료 확인**: 초록색 "서명이 적용되었습니다".
8. **주의사항**: 필수 서명 항목은 완료해야 제출됩니다.
9. **오류 해결**: 서명이 안 그려지면 화면을 새로고침하거나 이미지 업로드로 대체.
10. **FAQ**: [서명].

### 6-3. 파일 업로드 / 파일 다운로드
1. **기능 설명**: 증빙 서류를 **업로드**하고, 사업단이 제공한 양식을 **다운로드**합니다.
2. **대상**: 신청자. 3. **접근 경로**: 신청서 파일 항목.
4. **사전 확인**: 업로드 파일은 PDF·JPG·PNG·WEBP.
5. **단계별**: (1) 업로드 영역에 파일을 끌어다 놓거나 클릭해 선택합니다. (2) 안내 문구가 있는 항목은 확인창에서 **[확인]**. (3) **파일 다운로드** 항목은 링크를 눌러 양식을 내려받아 작성 후 다시 업로드 항목에 첨부합니다.
6. **입력 항목**: 파일(PDF·JPG·PNG·WEBP).
7. **완료 확인**: 업로드된 파일명이 목록에 표시.
8. **주의사항**: 허용 형식 외 파일은 거부됩니다. 재학증명서 등은 직인 날인본을 요구할 수 있습니다.
9. **오류 해결**: "이미지 또는 PDF만 업로드" → 형식 변환 후 재시도. 업로드 실패 시 다시 로그인 후 재시도.
10. **FAQ**: [오류 해결].

### 6-4. 표·근무상황부·비용 항목
1. **기능 설명**: 표 입력, 근무상황부(근로장학금), 교통비·등록비·숙박비 등 비용 항목을 입력합니다.
2~4. 신청자 / 신청서 해당 단계 / 프로그램에 따라 노출.
5. **단계별**: 표는 빈 칸을 채웁니다. 근무상황부는 날짜·시작·종료를 입력해 **[+ 등록]** 하면 근무시간이 자동 합산됩니다(수업시간과 겹치면 등록 불가). 비용은 항목별 금액·증빙을 입력합니다.
6. **입력 항목**: 표 셀, 근무 일자·시간, 비용 금액·증빙.
7. **완료 확인**: 합계 시간/금액 자동 계산 표시.
8. **주의사항**: 근무는 1일 최대 8시간·수업시간 제외. 숙박비 한도(1인 7만원) 등 안내 준수.
9. **오류 해결**: "종료 시간이 시작보다 늦어야" 등 안내 확인.

## 7. 성적 우수 지원금 — 교과목·평점 입력

![성적 우수 교과목 입력](images/apply-grade-detail.png)

1. **기능 설명**: 마이크로디그리/부전공/복수전공 이수 교과목과 성적을 입력하면 평점·인정학점·지급액이 자동 계산됩니다.
2. **사용 대상**: 성적 우수 지원금 신청 학생.
3. **접근 경로**: 지원금 신청 → 성적 우수 지원금.
4. **사용 전 확인사항**: 이수 교과목·성적, MD 이수 여부, (부전공/복수전공) 미래융합가상학과 이수(예정)자 여부.
5. **단계별 사용 방법**
   1. **세부 유형**(마이크로디그리 30만원 / 부전공 100만원 / 복수전공 150만원)을 선택합니다.
   2. (마이크로디그리) 학과 → MD 과정을 선택하고 교과목 성적을 입력합니다.
   3. (부전공/복수전공) 전공을 선택하고 **[+ 과목 추가]** 로 교과목을 추가합니다.
   4. 각 과목의 **학점·성적**을 선택합니다. MD 이수과목이면 **「MD 과정」** 을 지정합니다.
   5. 교과목 목록에 없으면 드롭다운에서 **✏️ 직접입력** 을 선택하고 **교과목명을 직접 입력**합니다.
   6. **지원 자격 확인** 체크리스트가 모두 충족되는지 확인합니다.
6. **입력해야 하는 항목**: 세부 유형, 전공/학과·MD 과정, 교과목명(선택 또는 직접입력), 학점, 성적, MD 지정.
7. **처리 완료 후 확인할 사항**: 평점 평균(3.0 이상), 인정 학점(기준 충족), 지급 예정액.
8. **주의사항**: '가/부' 과목은 평점 계산 제외. MD는 4과목 중 3과목만 인정 → 불인정 과목은 **[학점 불인정]** 체크.
9. **오류 발생 시 해결 방법**: "평점 3.0 미만" / "인정 학점 부족" → 교과목·성적 확인. "MD 과정에 포함되지 않은 교과목" 경고 → MD 지정 또는 교과목 재확인(직접입력 과목은 MD 지정하지 않음).
10. **관련 FAQ**: [대여 신청→성적 우수].

## 8. 임시저장 · 이어서 작성

1. **기능 설명**: 작성 중인 신청서를 저장하고 나중에 이어서 작성합니다.
2. **사용 대상**: 신청자. 3. **접근 경로**: 신청서 하단 **[임시저장]** / 마이페이지 임시저장 목록.
4. **사용 전 확인사항**: 로그인 상태.
5. **단계별 사용 방법**
   1. 신청서 작성 중 하단 **[임시저장]** 을 클릭합니다.
   2. 나중에 **마이페이지 → 임시저장(작성 중)** 에서 해당 건의 **[이어서 신청]** 을 클릭합니다.
   3. 저장된 내용이 복원되며, 마지막에 **제출**까지 완료합니다.
6. **입력해야 하는 항목**: 없음(현재까지 작성분 저장).
7. **처리 완료 후 확인할 사항**: "임시저장됨 HH:MM" 표시. 마이페이지 임시저장 목록에 노출.
8. **주의사항**: **임시저장은 접수가 아닙니다.** 반드시 제출해야 접수됩니다.
9. **오류 발생 시 해결 방법**: "임시저장 실패" → 다시 로그인 후 재시도.

## 10. 신청 상태 확인 · 승인/반려/보완요청

![마이페이지 신청 현황](images/mypage.png)

1. **기능 설명**: 마이페이지에서 신청별 검토·지급 상태와 진행 단계를 확인하고, 보완요청 시 수정·재제출합니다.
2. **사용 대상**: 신청자. 3. **접근 경로**: 상단 [마이페이지] (`/mypage`).
4. **사용 전 확인사항**: 로그인.
5. **단계별 사용 방법**
   1. **마이페이지** 로 이동합니다.
   2. 신청 카드에서 **검토 상태**(대기/검토중/승인/반려/보완요청 등)와 **지급 상태**를 확인합니다.
   3. **보완요청**이면 상단 배너/카드의 안내를 확인하고 **[수정 후 재제출]** 을 클릭해 수정 후 다시 제출합니다.
   4. **반려**면 안내(사유)를 확인합니다.
   5. 지원신청이 **승인**되면 **[지원금 신청하기]** 로 이어서 신청합니다.
6. **입력해야 하는 항목**: (보완요청 시) 수정 내용.
7. **처리 완료 후 확인할 사항**: 상태 배지·진행 단계 갱신. 최종 승인액(지원금).
8. **주의사항**: 상태는 사업단 검토에 따라 갱신됩니다. 본인 계정에서만 조회됩니다.
9. **오류 발생 시 해결 방법**: 상태가 안 바뀌면 **[새로고침]** 을 누르세요. 관리자 알림이 있으면 안내(메모)를 확인합니다.
10. **관련 FAQ**: [신청 상태], [승인/반려].

## 12. 내 정보 관리

마이페이지에서 관리합니다(수정 전 **비밀번호 본인 확인**).

- **개인정보 수정**: 이름·소속·연락처·이메일·계좌(학번 제외). 저장 시 신청에 자동 반영.
- **비밀번호 변경**: 새 비밀번호(6자↑)+현재 비밀번호 → 변경 후 재로그인.
- **학적상태변경**: 대학원 진학 등으로 학번이 바뀌어도 **기존 신청 기록 유지**. 학번 변경 시 새 학번으로 재로그인.
- **수강 시간표(근로장학금)**: 시간표를 입력하면 근무상황부에서 수업시간과 겹치는 근로 등록이 차단됩니다.

각 기능 공통 형식:
1. 기능 설명(위) 2. 대상: 본인 3. 경로: 마이페이지 해당 카드 4. 사전 확인: 현재 비밀번호
5. 단계별: (1) 카드 클릭 → (2) 비밀번호 본인 확인 → (3) 내용 수정 → (4) **[저장/변경]**
6. 입력 항목: 각 항목값 + 현재 비밀번호 7. 완료 확인: "저장/변경되었습니다"
8. 주의: 학번은 변경 불가(학적상태변경에서 별도). 9. 오류: 비밀번호 불일치 시 재입력.
10. FAQ: [로그인/계정].

## 14. 공간대여 신청

![공간대여 신청](images/space-rental.png)

2. **사용 대상**: 누구나(로그인 불필요).
3. **접근 경로**: 홈 → 공간대여, URL `/space-rental`.
4. **사용 전 확인사항**: 예약 캘린더에서 원하는 장소·시간이 비어 있는지 확인.
5. **단계별 사용 방법**

   2. (시간 항목) 필요 시 **종일**을 선택할 수 있습니다.
   3. 개인정보 수집·이용에 동의합니다.
   4. **[공간대여 신청]** 을 클릭합니다.
6. **입력해야 하는 항목**: 대여 장소, 날짜/시간, 신청자 이름·연락처, 사용 목적·인원 등(폼 설정에 따름), 동의.

8. **주의사항**: 이미 **신청·예약된 장소·시간과 겹치면 신청이 제한**됩니다. 다른 시간을 선택하세요.
9. **오류 발생 시 해결 방법**: "이미 신청·예약된 장소·시간대입니다" → 캘린더 확인 후 시간 변경. "종료 일시가 시작보다 늦어야" → 시간 재확인.
10. **관련 FAQ**: [대여 신청→공간대여].
